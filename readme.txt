== Description ==

If your Wordpress website uses Zawgyi to write posts, users cannot search with Unicode, vice versa.
With MM Search, users can search with both encoding formats.

== Installation ==

1. Upload `mm-search` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "MM Search Settings" at your Admin panel to configure settings.
